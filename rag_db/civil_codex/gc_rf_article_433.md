  - Общий принцип:
    - договор считается заключенным **в момент получения оферты акцепта лицом**, 
      направившим оферту  

  - Особые случаи:
    - если законом требуется **передача имущества** для заключения договора:  
      - договор считается заключенным с момента передачи имущества (ст. 224)  

    - если договор подлежит **государственной регистрации**:  
      - для третьих лиц договор считается заключенным с момента регистрации, если иное не установлено законом  

  - Практическое значение:
    - момент заключения договора определяет:
      - наступление прав и обязанностей сторон  
      - сроки исполнения обязательств  
      - ответственность за нарушения

# Статья 433 ГК РФ  
## Момент заключения договора

- Договор считается заключенным:
  - **общий случай**: при получении оферты акцепта  
  - **с передачей имущества**: с момента передачи соответствующего имущества  
  - **с регистрацией**: с момента государственной регистрации (для третьих лиц)
