# Статья 432 ГК РФ  
## Основные положения о заключении договора

- Договор считается заключенным, если:
  - достигнуто соглашение  
  - соблюдена форма  
  - согласованы **существенные условия**  

- Существенные условия:
  - предмет договора  
  - условия, установленные законом  
  - условия, необходимые для данного вида договора  
  - условия по требованию стороны  

- Механизм:
  - оферта → акцепт  

- Договор НЕ считается заключенным, если:
  - отсутствует согласие по существенным условиям  

- Ограничение:
  - нельзя ссылаться на незаключенность:
    - при принятии исполнения  
    - если это нарушает принцип добросовестности
