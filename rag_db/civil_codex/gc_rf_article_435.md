# Статья 435. ГК РФ  
## Оферта

- **Что такое оферта**: предложение конкретным лицам заключить договор с определёнными существенными условиями.  

- **Обязательность оферты**: связывает направившее её лицо с момента получения адресатом.  

- **Существенные условия**: должны быть указаны в оферте для возможности заключения договора.  

- **Отзыв оферты**:
  - если уведомление об отзыве дошло до адресата раньше или одновременно с офертой, оферта **не считается полученной**.  

- **Примечание**: коммерческое предложение признаётся офертой только при выполнении вышеуказанных условий.
